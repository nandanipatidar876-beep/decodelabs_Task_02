# decodelabs_Task_02
Project 02: Expense Tracker

Description

The Expense Tracker is a simple Python program that allows users to record expenses by entering the expense name and amount. The program displays all recorded expenses and calculates the total expenditure, helping users manage and track their spending efficiently.

Features

- Add multiple expenses
- Record expense names and amounts
- View expense summary
- Calculate total expenditure

Technologies Used

- Python 3
